## 数据类型


#### 基础数据类型
 - 数值型 
   - 整数类型 （byte,short,int,long）
 - 字符型（char）
 - 布尔型（boolean）

#### 引用数据类型
- 类
- 接口
- 数组

注意： 比如 String 类型也是类类型。

### 特点
一、从概念方面来说
	基础类型：变量名指向具体的数值
	引用类型：变量名指向存数据对象的内存地址
二、从内存构建方面来说
	基础类型：变量在声明之后java就会立刻分配给他内存空间
	引用类型：变量声明时不会分配内存，只是存储了一个内存地址
三、从使用方面来说
	基础类型：使用时需要赋具体值，判断时使用“==”号
	引用类型：使用时可以赋null 判断时使用equals方法
